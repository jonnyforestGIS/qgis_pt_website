    • PHP5-gd

Criar novo projecto no Lizmap

  1. Criar pasta dentro do Lizmap para conter todos os dados do projecto
  2. Carregar o Lizmap no browser e efectuar Login
    1. Admin > Your account > Lizmap configuration
    2. Create Respository
      • id: (sem espaços e caracteres especiais)
      • label
      • path: ../install/nome_da_pasta/ (apenas um exemplo)
